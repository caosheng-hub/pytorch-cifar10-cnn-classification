# PyTorch实现CIFAR10图像分类（CNN）
[![Python Version](https://img.shields.io/badge/Python-3.8+-blue.svg)](https://www.python.org/)
[![PyTorch Version](https://img.shields.io/badge/PyTorch-1.10+-red.svg)](https://pytorch.org/)

## 项目简介
本项目基于PyTorch框架实现CIFAR10数据集的图像分类任务，采用经典CNN架构（参考LeNet-5改进），聚焦**模块化代码设计、训练流程标准化、结果可解释性**，适合深度学习入门与工程化实践。
> 该项目为计算机视觉/算法岗实习求职的基础实战项目，覆盖「数据处理→模型搭建→训练调优→评估分析」全流程。

## 技术栈
- 框架：PyTorch（核心）、Torchvision
- 工具：Matplotlib（可视化）、torchsummary（模型结构分析）、time（性能统计）
- 算法：CNN、交叉熵损失、Adam优化器、MaxPooling、ReLU激活

## 项目亮点
1. **模块化设计**：将数据加载、模型定义、训练、评估解耦，符合工业界代码规范；
2. **全流程监控**：训练过程实时统计损失/准确率/耗时，评估阶段输出整体精度；
3. **可扩展性强**：支持快速替换模型结构、优化器、数据增强策略；
4. **结果可解释**：附带训练日志和性能分析，体现问题分析能力。

## 数据集说明
CIFAR10包含10类RGB彩色图像（32×32），共60000张：
- 训练集：50000张
- 测试集：10000张
- 类别：飞机、汽车、鸟类、猫、鹿、狗、青蛙、马、船、卡车

## 模型结构
| 层类型       | 参数配置                          | 输出维度（输入3×32×32） |
|--------------|-----------------------------------|------------------------|
| 卷积层1      | 3→6通道，3×3卷积核，步长1，无填充 | 6×30×30                |
| 最大池化层1  | 2×2窗口，步长2                    | 6×15×15                |
| 卷积层2      | 6→16通道，3×3卷积核，步长1，无填充| 16×13×13               |
| 最大池化层2  | 2×2窗口，步长2                    | 16×6×6                 |
| 全连接层1    | 576→120维度                       | 120                    |
| 全连接层2    | 120→84维度                        | 84                     |
| 输出层       | 84→10维度（分类数）               | 10                     |
