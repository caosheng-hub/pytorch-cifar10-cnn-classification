
### 📌 项目名称

基于 PyTorch 的 CIFAR-10 图像分类系统（CNN）

---

### 📌 项目描述

基于 PyTorch 深度学习框架，从零实现卷积神经网络（CNN）模型，对 CIFAR-10 数据集进行多分类任务建模。项目覆盖数据加载、模型设计、训练优化、性能评估及模型持久化，构建完整的计算机视觉任务流程，并对模型结构及训练策略进行一定优化探索。

---

### 📌 技术栈

Python / PyTorch / torchvision / NumPy / Matplotlib

---

### 📌 核心工作

#### 🧩 1. 数据处理与加载

* 基于 `torchvision.datasets` 构建 CIFAR-10 数据集加载流程
* 使用 `DataLoader` 实现批量数据加载（Batch Size=8）及随机打乱（Shuffle）
* 完成张量化（ToTensor）及基础预处理流程
* 明确训练集（50k）与测试集（10k）划分

---

#### 🧠 2. 模型设计（CNN）

* 独立设计轻量级卷积神经网络结构：

  * 2 层卷积层（提取空间特征）
  * 2 层最大池化（降维 + 防过拟合）
  * 3 层全连接层（分类决策）
* 使用 ReLU 激活函数增强非线性表达能力
* 手动完成特征展平（Flatten）并匹配全连接输入维度（576）
* 使用 `torchsummary` 对模型结构进行可视化验证

---

#### ⚙️ 3. 模型训练与优化

* 构建完整训练流程（forward → loss → backward → step）
* 使用 `CrossEntropyLoss` 处理多分类问题（Softmax + NLL）
* 选用 Adam 优化器进行参数更新（lr=1e-3）
* 实现 epoch 级别训练循环（10 epochs）
* 自定义统计指标：

  * 平均 Loss
  * 分类准确率（Accuracy）
* 记录每轮训练耗时，分析训练效率

---

#### 📊 4. 模型评估与验证

* 切换模型至 `eval()` 模式进行测试
* 基于 `argmax` 获取预测类别
* 在测试集上计算整体准确率
* 验证模型泛化能力

---

#### 💾 5. 模型持久化与推理

* 使用 `torch.save` 保存模型参数（.pth 文件）
* 使用 `load_state_dict` 完成模型加载
* 构建完整推理流程，实现训练→加载→预测闭环

---
